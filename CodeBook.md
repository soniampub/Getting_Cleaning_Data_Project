# Getting_Cleaning_Data_Project
# Code Book
This Codebook summarizes the data fields in "tidy.txt" file which is end result of running "run_analysis.R" script on our testing and training data.

## Identifiers
* subject : The id of the subject
* activity : Type of activity performed by subject

## Activity Lables
* SITTING : Subject was sitting when test was performed.
* STANDING : Subject was standing.
* WALKING : Subject was walking.
* WALKING_UPSTAIRS : Subject was walking upstairs.
* WALKING_DOWNSTAIRS : Subject was walking downstairs.
* LAYING : Subject was laying.
